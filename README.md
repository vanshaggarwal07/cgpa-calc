# CGPA Calculator

IPU CGPA Calculator - A modern web application for calculating CGPA for IP University students (5th semester CSE, IT, and ECE branches).

## Features

- Calculate CGPA based on marks and credits
- Support for multiple branches (CSE/IT and ECE)
- Modern, attractive UI design
- Real-time calculation
- Beautiful gradient design with smooth animations
